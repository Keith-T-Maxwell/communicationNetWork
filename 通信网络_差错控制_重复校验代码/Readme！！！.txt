该差错校验代码支持将任意长度的文本转化为二进制。以及重复校验转换后的二进制流。
默认转换为原来的三倍长。
注意！该代码不包含heading和ending！！！ 如果要转换heading和ending码为原来
的三倍长，请你自行手动添加。

如果遇到数组过长，无法粘贴的问题，请看word文本中的教程内容。
